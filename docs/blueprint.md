# **App Name**: EduPlanner

## Core Features:

- Create Subject: Allow users to create new subjects with details such as title, description, and related topics.
- List Subjects: Display an interactive and filterable list of all subjects.
- Update Subject: Enable users to update the details of existing subjects.
- Delete Subject: Allow users to delete subjects from the database.
- Intelligent topic suggestion: Suggest new topics based on what subjects are already listed.
- Topic Hierarchy: Represent topics of each subject in a hierarchy with numbered topics in a sidebar.
- CRUD Topics: Allow Create, Read, Update, and Delete operations on each topic.
- Topic Completion Status: Allow users to mark topics as complete or incomplete.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) to convey professionalism and trust.
- Background color: Light grey (#E0E0E0) for a clean and modern interface.
- Accent color: A slightly brighter blue (#3D84A8) to highlight interactive elements.
- Body and headline font: 'Inter' sans-serif font, providing a clean and neutral reading experience.
- Use minimalist icons to represent actions and subject categories.
- Employ a clean and structured layout for easy navigation.
- Use subtle animations for interactive feedback.
